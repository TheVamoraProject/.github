## ⚡ What is Vamora?

Vamora is a multi-platform software ecosystem focused on crafting modern apps, experiments, and tools that actually feel good to use.

We build things that live across:
- 🌐 Web
- 📱 Mobile
- 💻 Cross-platform apps
- ⚙️ Experimental systems & tools
